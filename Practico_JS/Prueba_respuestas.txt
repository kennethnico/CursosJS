/***Funciones**/
1.-
a) Una función es un bloque de código que se puede reutilizar en nuestro proyecto.
b) Cuándo fragmentos de nuestro proyecto se repiten, podemos dividirla y reutilizar la función, en lugar de reescribir el código.
c) Argumento: El valor que se asigna al "llamar" a una función.
    Parámetro: Propiedad especificada al definir una función.

/**Condicionales**/
1-.
a) Una condicional, es una serie de instrucciones que se realizan dependiendo del resultado de una premisa con resultado boolean. (Si esto, entonces)
b) Los tipos de condicional que existen son If-Else, If...for y Switch
   If-else: Si se cumple la condición, entonces...
   If...for: Es una condicional ternaria, una manera compacta de If.
   Switch: Evalúa por casos la condición.
c) Si se puede combinar funciones y condicionales.

/**Ciclos**/
a) Un ciclo es un segmento de código que se ejecuta la cantidad de veces que sea necesario.
b) En JS existen dos tipos de ciclos: for y while
c) Un ciclo infinito es un bloque de código que se ejecutará de manera indefinida, el problema con este tipo de ciclos es que llegan a consumir mucha memoria y entorpecer el buen funcionamiento, no solo del código, sino también del navegador.
d) Si, se pueden combinar los ciclos y condicionales.

/*Listas*/
a) Un array es un objeto similar a una lista con características que permite cambiar/editar sus elementos.
b) Un objeto es una entidad que posee propiedades independientes.
c) Podríamos usar una array en, por ejemplo, una lista de autos y un objeto para definir las propiedades de esa lista de autos, especificando la marca, precio, modelo, color, etcétera.
d) Sí se pueden combinar.
